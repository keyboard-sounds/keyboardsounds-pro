Keyboard Sounds Pro

Type. Click. Sound.

A free desktop app that plays customizable keyboard and mouse sounds while you work. Make every keystroke satisfying.

Download on GitHub Join Discord

Free & open source · 20+ built-in profiles · Windows, Linux

Features

Application Rules

Control the sound daemon based on the currently focused application. Enable only in your text editor or terminal, disable for specific games, or localize sounds to a particular app.

Windows only

On-Screen Modifiers

Display active keyboard shortcuts and modifiers on screen as you type. Perfect for streamers, content creators, and anyone who wants to show their key presses visually.

Windows only

Audio Effects

Pitch Shift — Change the pitch within a range. Pan — Pan left or right. Equalizer — Adjust frequency response to your liking.

Custom Profiles

Support for both Mouse and Keyboard audio profiles. Comes bundled with twenty built-in profiles (13 keyboard, 7 mouse). Supports custom profiles through the editors (.wav and .mp3), and import/export to share with others.

Profile Builder

Build your own profiles using audio files you provide. Supports both keyboard and mouse profiles. Share them with others and import into the app.
