## Privacy Policy

Keyboard Sounds Pro respects your privacy and collects only minimal, non-personal information required to understand general app usage.

The application records Daily Active Users (DAU) for the purpose of measuring how many systems use the app each day. To do this, the app generates a random identifier unique to each installation. This identifier contains no personal information and cannot be used to identify you.

Keyboard Sounds Pro does not collect, store, or transmit any personally identifiable information, including but not limited to names, email addresses, IP addresses, or system files.

The collected usage data is used solely to understand overall application usage and improve the software.

No data collected by Keyboard Sounds Pro is sold, shared, or used for advertising.

By using Keyboard Sounds Pro, you agree to the collection of this minimal anonymous usage information.
